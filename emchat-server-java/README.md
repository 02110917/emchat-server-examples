## 环信服务器端实例代码

本项目使用[Gradle](http://gradle.org)进行构建, 可以直接导入到IDE中, 例如

### 导入到eclipse中:

在命令行中运行

    gradlew.bat eclipse
    
这个命令会生成eclipse的项目文件, 然后可以通过eclipse的import工具导入

### 导入到Intellij中:

在命令行运行

    gradlew.bat idea
    
然后直接在Intellij中打开这个项目

### 直接编译

在命令行中运行 

    gradlew.bat clean compile
